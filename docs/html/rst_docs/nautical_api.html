nautical_api.connector module

This file will act as the connector script that will be used to pull all of the data using the nautical library and put the information into a database

nautical_api.connector.find_wait_time(t=None)

Use the current time (minutes) passed the hour to determine the wait time until the next expected period. This function should only be needed once as each interval will be 30 minutes after the last lookup.

Parameters

t – Time in minutes currently past the hour, When None use datatime

class nautical_api.resources.AllBuoysGetter

Bases: flask_restful.Resource

The class implements the ability or resource that will GET all buoys. Each buoy contains the current information registered at the buoy. The current information will vary based buoy. The information is updated roughly twice/hour. Please see nautical.noaa.buoy.buoy_data for more information.
